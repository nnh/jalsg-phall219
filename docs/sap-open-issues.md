  - なお、2026-06-22時点のメールでは一旦「PRT記載通りCHR/CHRp到達例のみ（#49除外）」で収束したやり取りが残っているが、その後の症例検討会にて上記の通り広義解釈（#49を含める）へ最終的に変更された。本ファイルは2026-07-20の最終決定を正とする。
