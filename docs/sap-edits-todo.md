## 1. §4.4.3 無再発生存時間（RFS）— 解析対象集団の決定理由の明記【2026-07-20最終決定】

**内容**：解析対象集団の定義そのもの（広義、LFSのみ到達例＝#49を含む）は**変更しない**。2026-06-22時点のメールでは一旦「PRT記載通りCHR/CHRp到達例のみ（#49除外）」で収束していたが、2026-07-20の症例検討会にて、CRi文献動向を理由に「#49を含める（広義）」で最終決定した。この決定理由と#49の帰属を、監査証跡として注記に明記する。

**現行文**：
```
解析対象集団：FASのうちCHR到達例（CHRpおよびLFSのみ到達例を含む）
　　　　（RS.RSTESTCD='HEMARESP' かつ RSORRES IN ('LFS','CHRp','CHR')
　　　　かつ VISITNUM IN (310, 410) のレコードを有する症例）
注：研究計画書「8.1.11.3. 無再発生存期間 (relapse-free survival, RFS）」において、「CHRに到達患者を対象とする」と規定しているが、同項にて起算日は「白血病細胞消失（LFS）到達日」と定義している。また、研究計画書「8.1.4. 血液学的効果判定の基準」において、LFS は「骨髄芽球≤5%かつ髄外病変なし」を満たす白血病細胞消失状態であり、血液学的回復が不完全なcomplete remission with incomplete count recovery（CRi）に相当する。本試験においては、プロトコルの起算日定義と整合する形で、LFSのみ到達例も「CHR到達例（広義）」としてRFSの解析対象に含める。なお、論文化に際しては「CHR 到達例（CRi に相当する LFS 単独到達例を含む）」と記載する。
起算日：白血病細胞消失（LFS）到達日
       （RS.RSTESTCD='HEMARESP', RSORRES IN ('LFS','CHR','CHRp'),
        VISITNUM IN (310, 410) のうち最初に記録された評価日）
```

**修正後文（案・現行の定義に決定理由の注記を追記）**：
```
解析対象集団：FASのうちCHR到達例（CHRpおよびLFSのみ到達例を含む）
　　　　（RS.RSTESTCD='HEMARESP' かつ RSORRES IN ('LFS','CHRp','CHR')
　　　　かつ VISITNUM IN (310, 410) のレコードを有する症例）
注：研究計画書「8.1.11.3. 無再発生存期間 (relapse-free survival, RFS）」において、「CHRに到達患者を対象とする」と規定しているが、同項にて起算日は「白血病細胞消失（LFS）到達日」と定義している。また、研究計画書「8.1.4. 血液学的効果判定の基準」において、LFS は「骨髄芽球≤5%かつ髄外病変なし」を満たす白血病細胞消失状態であり、血液学的回復が不完全なcomplete remission with incomplete count recovery（CRi）に相当する。本試験においては、プロトコルの起算日定義と整合する形で、LFSのみ到達例も「CHR到達例（広義）」としてRFSの解析対象に含める。なお、論文化に際しては「CHR 到達例（CRi に相当する LFS 単独到達例を含む）」と記載する。
起算日：白血病細胞消失（LFS）到達日
       （RS.RSTESTCD='HEMARESP', RSORRES IN ('LFS','CHR','CHRp'),
        VISITNUM IN (310, 410) のうち最初に記録された評価日）
【症例検討会決定事項（2026-07-20）】実データ上、LFSのみ到達しCHR/CHRp未到達の症例は#49の1例のみである。研究計画書8.1.11.3を文字通り読めば#49はRFS解析対象外となるが、LFSはCRi（complete remission with incomplete count recovery）に相当する状態であり、近年の文献ではCRi（≒LFS）を含めてCRとして解析する例が多いという臨床的動向を踏まえ、統計解析責任者の判断として#49をRFS解析対象に含めることを症例検討会にて決定した。
```

## 2. §5.2.5 移植後TKI投与（治療投与・予防投与）— 直近MR未検査時の扱いの修正

**現行文**（表脚注部分）：
```
＊移植後のMRがNDの場合は「予防投与」、NQ/DTの場合は「治療投与」として扱う。移植後TKI投与の治療/予防区分は、移植後TKI開始日の直近のMR評価を用いる。
移植後TKI開始日：EC.ECSPID='after_transplantation' かつ ECTRT IN ('DASATINIB HYDRATE','PONATINIB HYDROCHLORIDE') かつ ECSTDTCが空でないレコードのうち最小のECSTDTC。
直近MR評価：LB.LBSPID IN ('evaluation30','evaluation60','evaluation100') かつ LBTESTCD='MJBCRABL'（Major BCR-ABL1, 患者のBCR-ABL1サブタイプがminorの場合は'MNBCRABL'）かつ LBSTAT≠'NOT DONE'のうち、TKI開始日（ECSTDTC）直前のLBDTCのレコード。
判定：
- LBORRES = "0" → ND（検出不能）→ 「予防投与」
- 0 < LBORRES ≤ 50 → NQ（≤50 copies/μgRNA）→ 「治療投与」
- LBORRES > 50 → DT（>50 copies/μgRNA）→ 「治療投与」
- LBORRESの値が存在しない場合（未検など）→「予防投与」として取扱う
※(NQ)≦50は49, (ND)検出不能は0が入力されている。
※BCR-ABL1サブタイプ（MJBCRABL/MNBCRABL）の判定は、診断時のMB.csv または LB.csvベースライン記録から取得する。（症例ごとにMJBCRABLまたはMNBCRABLのどちらを採用するか決定）
TKI開始日直前のLB評価が存在しない（または LBSTAT='NOT DONE' のみで定量値が未取得の）場合は、保守的に「予防投与」として取扱う。該当症例が複数発生した場合は集計表脚注に明示する。
```

**修正後文（案）**：
```
＊移植後TKI投与の治療/予防区分は、TKI開始日以前で直近に得られたMR評価（移植前後を問わない）に基づいて判定する。MRがND（検出不能）の場合は「予防投与」、NQ/DTの場合は「治療投与」として扱う。
移植後TKI開始日：EC.ECSPID='after_transplantation' かつ ECTRT IN ('DASATINIB HYDRATE','PONATINIB HYDROCHLORIDE') かつ ECSTDTCが空でないレコードのうち最小のECSTDTC。（変更なし）
直近MR評価：LB.LBSPID IN ('pre_transplantation','evaluation30','evaluation60','evaluation100')（移植前評価を含める）かつ LBTESTCD='MJBCRABL'（Major BCR-ABL1, 患者のBCR-ABL1サブタイプがminorの場合は'MNBCRABL'）かつ LBSTAT≠'NOT DONE'のうち、TKI開始日（ECSTDTC）以前で最も直近のLBDTCのレコード。
判定：（閾値は変更なし）
- LBORRES = "0" → ND（検出不能）→ 「予防投与」
- 0 < LBORRES ≤ 50 → NQ（≤50 copies/μgRNA）→ 「治療投与」
- LBORRES > 50 → DT（>50 copies/μgRNA）→ 「治療投与」
移植後すぐ（day30含む）にMR未検のままTKI投与を開始した症例は、移植前のMR評価に基づいて区分する。移植前MRがND/DTであった場合は「治療投与」として扱う（PI見解：「移植前のMRはND/DTになると思います。その場合はやはり『予防投与』より『治療投与』に意味合いは近い」2026-06-23）。
なお、当初案（TKI開始日直前のLB評価が存在しない場合は一律「予防投与」として取扱う）は撤回する。
※(NQ)≦50は49, (ND)検出不能は0が入力されている。（変更なし）
※BCR-ABL1サブタイプ（MJBCRABL/MNBCRABL）の判定は、診断時のMB.csv または LB.csvベースライン記録から取得する。（変更なし）
※#61は移植未到達のため本区分の対象外。
```

## 3. §5.2.5／§5.2.1図表脚注 — MR判定のオーツカ測定キット小数点対応【要PI確認・該当なし】

メール上に明示合意記録はないため、下記は「PRT記載に整合させる案」であり反映前にPI確認が必要。

**現状**：§5.2.5および§5.2.1の図表脚注はいずれも「LBORRES≤50→NQ、>50→DT」というcopies/µgRNAスケールの整数閾値のみで判定。オーツカキット固有の閾値（PRT §8.1.1.1：minor NQ<0.00151%、major NQ<0.0007%、%表記）への言及がない。一方、§5.2.1脚注※4は「オーツカ添付文書記載の変換式で変換した値を報告頂く」運用を記載しており、CRF入力時点で標準スケールへ変換済みという前提の可能性がある。

**提案する追記文（案、要PI・DM確認）**：
```
※オーツカ測定キットで測定された症例についても、施設は添付文書記載の変換式により標準スケール（copies/µgRNA相当）に変換した値をLBORRESとして報告する（§5.2.1※4参照）。データセット上、全例でこの変換が適用されていることをデータマネジメント担当に確認する。未変換の生値（%表記）が混入している場合は、以下の基準で別途判定する：
　minor: LBORRES<0.00151%→NQ、≥0.00151%→DT
　major: LBORRES<0.0007%→NQ、≥0.0007%→DT
```
**次アクション**：データマネジメント担当（生越さん）に、全例でLBORRESが変換済み値であるかを確認したうえで、この追記の要否をPIと確定する。

## 4. §4.5.2 早期死亡・治療関連死 — 3相別起算日への修正【要PI確認・該当なし】

メール上に明示合意記録はないため、下記はPRT §9.4.2.3〜9.4.2.4に整合させる案であり反映前にPI確認が必要。

**現行文**：
```
治療開始後14日以内の死亡（早期死亡）及び因果関係が否定できない死亡（治療関連死）の頻度と原因を解析する。
「治療開始日」は以下のいずれかと定義する。
(1) PSL先行療法が実施された症例: PSL先行療法開始日（EC.ECSPID='prephase' の最小ECSTDTC）
(2) PSL先行療法が実施されなかった症例: 寛解導入療法DA投与開始日（EC.ECSPID='induction' の最小ECSTDTC）
```

**修正後文（案）**：
```
早期死亡は、研究計画書9.4.2.3〜9.4.2.4の規定に従い、以下の3相それぞれについて、各治療開始日から14日（2週間）以内の死亡例数・割合・原因を算出する（対象：FAS）。
(1) 寛解導入療法：起算日はPSL先行療法実施例ではPSL先行療法開始日（EC.ECSPID='prephase'の最小ECSTDTC）、非実施例では寛解導入療法DA投与開始日（EC.ECSPID='induction'の最小ECSTDTC）。分母は寛解導入療法の一部以上が施行されたFAS症例。
(2) 強化地固め療法：起算日は強化地固め療法開始日（該当EPOCHの最小ECSTDTC）。分母は強化地固め療法の一部以上が施行された症例。
(3) PN導入された最初のコース：起算日はPN投与開始日（EC.ECTRT='PONATINIB HYDROCHLORIDE'の最小ECSTDTC）。分母はPNが導入された全患者。
治療関連死（因果関係が否定できない死亡）は、FAS全体を分母として別途算出する（現行記載通り、変更なし）。
```
**次アクション**：EC.ECSPIDの実際の値（強化地固めに対応するコード）をプログラム担当・DMに確認したうえで、上記案をPIに提示し最終確認する。
